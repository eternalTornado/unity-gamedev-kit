# Zenject / Extenject DI Profile

Overlay that configures Zenject (Extenject) as the project's dependency injection framework.

## What this adds
- `.claude/rules/di-zenject.md` -- Zenject coding rules and patterns
- Patches `CLAUDE.md` Technology Stack to reference Zenject

## Apply
```bash
ugk init --di zenject
# or add to existing project:
ugk add profile di-zenject
```
