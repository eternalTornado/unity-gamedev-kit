# VContainer DI Profile

Overlay that configures VContainer as the project's dependency injection framework.

## What this adds
- `.claude/rules/di-vcontainer.md` -- VContainer coding rules and patterns
- Patches `CLAUDE.md` Technology Stack to reference VContainer

## Apply
```bash
ugk init --di vcontainer
# or add to existing project:
ugk add profile di-vcontainer
```
